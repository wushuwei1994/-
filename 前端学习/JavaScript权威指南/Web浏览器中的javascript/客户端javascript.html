  • Window对象，客户端JavaScript特性的主要接入点，表示Web浏览器的一个窗口或窗体，可用标识符window来引用它
  • location为Window对象的属性，指Location对象，指定当前显示在窗口的URL，并允许脚本往窗口里载入新的URL
  • 可通过window.location = 'url'方式直接跳转到新的url地址
  • window定义的方法：alert()//提示窗口, setTimeout(handler,2000)//定时器
  • window对象的属性和方法实际上是全局变量和全局函数，可以不用到window而直接使用
  • document是Window对象中最重要的属性，它引用Document对象，表示显示在窗口中的文档
  • document方法：getElementById();//可以基于元素id属性的值返回单一的文档元素Element对象，Element对象有属性和方法
  • Element对象方法：appendChild();//拼接Node元素
    对象属性：firstChild,style,className
  • Window、Document和Element对象上一个重要的属性集合是事件处理程序相关的属性。可以在脚本中为之绑定一个函数，这个函数会在某个事件发生时以异步的方式调用。事件处理程序的属性名是以单词"on开始"
  • 事件相关属性：onclick,onload
  • Window对象的onload,当显示在窗口中的文档内容稳定并可以操作时(一般整个文档加载完成时，也包括文档最后script标签内容)会触发它。JavaScript代码通常封装在onload事件处理程序里
  • XMLHttpRequest对象，可以对HTTP请求 编程来启用网络，通常叫做Ajax
