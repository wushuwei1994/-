## MVC前端控制器

![img](img/6fe0d67d2add48ccaa9dbe3679ae11d4/clipboard.png)

访问请求进入前端控制器（Front controller），前端控制器把请求分发给相应控制层处理相关业务，并把处理结果返回给前端控制器，前端控制器把数据装载在视图模板以页面呈现形式返回给客户端

MVC

M：model，数据模型

V：view，页面视图

C：controller，控制层，调用业务逻辑产生合适的数据（model），传递数据给视图层用于呈现。
