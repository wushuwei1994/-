##### 安装Java JDK8（开源版本）

1、更新软件包列表：

```bash
sudo apt-get update
```

2、安装openjdk-8-jdk：

```bash
sudo apt-get install openjdk-8-jdk
```

3、查看java版本，看看是否安装成功：

```bash
java -version
```
