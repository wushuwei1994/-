## 基础

命令格式
		command [-options] parameter1 parameter2 ...
		命令     -选项      参数1       参数2...
		说明：
		1. 命令、选项、参数等这几个命令中间以空格（多个空格也只当一个空格）来区分
		2. 通常参数前会带有 - 号，使用完整参数名时，需要带 -- 符号。
		3. 反斜杠(\)来转义[Enter]符号，使命令连续到下一行。


常用热键

* [Tab]按键 命令（文件）补全提示 
* [Ctrl]-c按键 中断正在运行的命令 
* [Ctrl]-d按键 结束输入 




在命令执行中，反单引号\[`\](按键1左边的键)之内的命令将会被先执行，而其执行出来的结果将作为外部的输入信息。
